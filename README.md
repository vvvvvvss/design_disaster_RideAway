# Design Disaster - Cab Booking App

A React + Vite project that intentionally violates UI/UX principles to demonstrate design flaws. Built for the Design Disaster competition with strategic, clever design disasters.

## Project Structure

- `src/` - React components and styles
  - `App.jsx` - Main app component with navigation
  - `components/` - Individual feature components:
    - `Navigation.jsx` - Intentionally confusing navigation
    - `RestaurantList.jsx` - Misleading restaurant listings
    - `MenuBrowser.jsx` - Confusing menu layout
    - `Checkout.jsx` - Hidden and confusing checkout
    - `OrderTracking.jsx` - Difficult order tracking

## Features (UX Disasters)

1. **Confusing Navigation** - Uneven sizes, hidden options, non-intuitive layout
2. **Misleading Ratings** - Colors used incorrectly (red for high rating, green for low)
3. **Hidden Checkout** - Checkout button only visible on hover, nearly invisible
4. **Poor Menu Layout** - Items scattered, inconsistent sizing, overlapping text
5. **Misleading Discounts** - Complex disclaimers, unclear terms
6. **Confusing Pricing** - Hidden fees, unclear totals, random "chaos surcharges"
7. **Difficult Order Tracking** - Multiple conflicting statuses, unclear progress
8. **Bad Aesthetics** - Clashing colors, poor contrast, animated elements
9. **Uneven Typography** - Random font sizes throughout the interface
10. **Non-interactive Elements** - Buttons that look clickable but aren't, and vice versa

\`\`\`

\`\`\`

### Development Server
\`\`\`

\`\`\`
The app will run at `http://localhost:3000`

### Build for Production
\`\`\`
npm run build
\`\`\`

## Technologies

- React 18
- Vite (for fast development)
- CSS3 with animations and gradients

## Educational Purpose

This project is designed for the "Design Disaster" competition to demonstrate:
- Understanding of UI/UX principles by violating them intentionally
- Impact of design flaws on user experience and accessibility
- Common mistakes in web design and their consequences
- Importance of good design practices
